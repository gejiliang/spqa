# Annalista · 长期记忆 (Long-Term Memory)

## 内存结构 (Memory Structure)

**档案文件本身即为记忆** — archive/ 目录中的所有 YAML 前置元数据存档构成了组织的持久化记忆。

MEMORY.md 仅维护**元层面记忆**：

### 元级别追踪 (Meta-Level Tracking)

- **存档索引指数** (Archive Index): 按年月统计，每个月的存档数量与类型分布
- **重要决议清单** (Important Decisions): 标记为 high 重要性的决议链接与摘要
- **Caesar 倾好** (Caesar Preferences): 根据历史查询模式记录 Caesar 关注的主题
- **参与者档案** (Participant Profiles): 频繁参与决议的核心人物及其角色

### 检索优化 (Retrieval Optimization)

- 常见查询模式缓存
- 高频访问存档的快速路径指标
- 语义搜索权重调整记录
